# Star Safety Enterprises

Premium, industrial-grade safety net installation services across Tamil Nadu, India. Highly specialized in balcony safety, pigeon exclusion, bird protection, construction barriers, and sports net installations.

## ✨ Core Features

- **Interactive 3D Material Wheel**: A fluid, responsive vertical 3D carousel highlighting high-tensile safety net types and specifications.
- **Custom Estimate & Site Visit Booking**: A multi-step modal form with dynamic city validation supporting all major locations across Tamil Nadu.
- **Micro-Interactions & Premium Shimmers**: Smooth mouse hover states with dynamic scale adjustments, hardware-accelerated translational offsets, and custom glow backings for buttons.
- **Instant Connectivity**: Direct, responsive WhatsApp API integrations and instant call triggers.
- **Rich Resource Feed**: Informative guide modules highlighting optimal mesh sizes, knotless tension technology, and durability testing.
- **Adaptive Architecture**: Fully optimized layout designed with modular TypeScript components and styled using modern Tailwind CSS.

## 🚀 Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS v4, custom glassmorphism layers, fluid animations
- **Icons**: Lucide React
- **Backend Server**: Express (Node.js) with tsx/esbuild bundling for production
- **Deployment**: Configured for lightweight, lightning-fast container ingress on Cloud Run

## 📦 Getting Started

### Installation
Install the necessary project dependencies:
```bash
npm install
```

### Run in Development
Start the full-stack development server supporting hot-module reloading:
```bash
npm run dev
```

### Production Build
Generate a streamlined production package containing static assets and bundled backend modules:
```bash
npm run build
```

### Start Server
Run the compiled CommonJS server:
```bash
npm run start
```
